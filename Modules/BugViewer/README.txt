BugForge
========
A robust Lua error capture and viewer for World of Warcraft.

Features:
- Captures all Lua errors (with stack, time, count)
- Persistent storage across sessions
- UI for browsing, copying, clearing, and exporting errors
- Minimap button for quick access
- Slash commands: /bugforge, /errors
- Export and copy all errors at once
- Search/filter planned for future

Install:
- Place the BugForge folder in your AddOns directory.
- Enable in-game.
